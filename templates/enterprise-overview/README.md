# 企业运营如意数据大屏 · Enterprise Overview

> **RuyiBigScreen** 仓库的第一个正式模板：通用综合型企业运营大屏。
>
> 复制 `index.html` 单文件即可在浏览器双击打开，无需任何构建工具或本地依赖。

![preview](./preview.png)

## ✨ 模板特点

- 🎨 **科技深蓝风格**：深空蓝黑底 + 冰蓝霓虹渐变 + 动态粒子背景
- 📐 **黄金九宫格布局**：顶部通栏 + 左右三栏 + 中部地图 + 底部版权
- 📊 **9 个图表组件**：4 KPI 数字卡 + 双折线 + 中国地图 + 柱状 + 仪表盘 + 饼图 + 环形 + 雷达 + 横向柱 + TOP10 滚动榜单
- 🎬 **动效齐全**：数字滚动跳变、地图涟漪、榜单自动滚动（悬停暂停）、图表渐入、背景粒子浮动
- 🖥️ **超宽适配**：1920×1080 / 2560 / 3840 均能正常显示
- 🚀 **零依赖运行**：所有资源通过公共 CDN 引入，无 build 步骤

## 🚀 快速开始

```bash
# 直接双击打开
open index.html     # macOS
start index.html    # Windows
xdg-open index.html # Linux

# 或者起一个静态服务（推荐）
```

## 🛠️ 二次定制指南

### 1. 替换假数据

打开 `index.html`，搜索以下变量或数据点：

- `RANK_DATA`：TOP10 榜单人员
- `data-target` 属性：4 个 KPI 目标值
- 各 `initChart(...)` 调用里的 `data: [...]`：图表数据
- 中国地图的 `data: [...]`：省份销售额

### 2. 替换为你的业务数据

把 `RANK_DATA`、`data-target` 改为后端 API 返回值即可，例如：

```js
// 伪代码
fetch('/api/kpi').then(r => r.json()).then(d => {
  el.dataset.target = d.total;
  animateNumber(el, d.total, 1500);
});
```

### 3. 改主题色

全局搜索 `#00e5ff` 替换为你想要的主色，比如：

| 风格     | 主色         |
| -------- | ------------ |
| 蓝紫科幻 | `#7c5cff`    |
| 暗黑绿    | `#00ffa3`    |
| 警示橙    | `#ff8a00`    |
| 商务金    | `#ffd76a`    |
